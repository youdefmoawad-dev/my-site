ما هي CSS؟

CSS هي اللغة المسؤولة عن تنسيق وتصميم صفحات الويب مثل الألوان، الخطوط، الأحجام، وتنسيق العناصر.

ماذا ستتعلم؟

  • الألوان والخطوط
  • التخطيطات (Layout)
  • Flexbox و Grid
  • تصميم متجاوب
